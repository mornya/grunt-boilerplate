## About
The development environments for static resources by Grunt.js based.
> Grunt를 이용한 www 빌드용 boilerplate (~2016.12)

## Execution
아래와 같이 커맨드 라인에서 CLI를 실행하여 로컬 서버를 실행한다.
```bash
$ grunt
```
